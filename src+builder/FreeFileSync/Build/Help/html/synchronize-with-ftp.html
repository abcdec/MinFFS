Synchronize with SFTP (Windows)

FreeFileSync support synchronization with SFTP natively.

Note
In case the FTP server sets file modification times to the current time you can do a Compare by File Size as a workaround.


Synchronize with SFTP (Linux)

An SFTP share can be easily mapped onto a local folder for use with FreeFileSync:

  • Install:
    sudo apt-get install sshfs

  • Mount SFTP share:
    sshfs ssh-account@ssh-server:<path> mountpoint

  • Unmount:
    fusermount -u mountpoint
